# Movie test

## Tech requirements
- React v17.0.2
- Node v14.17.0 LTS
- npm v6.14.11

## Running the app
```
nvm install 14.17.0
```

```
nvm use 14.17.0
```

```
git clone https://github.com/monbird/movie-test.git
```

```
cd movie-test
```

```
npm install
```

```
npm start
```
